### 🤝💼📊 Meetora

Meetora is a modern real-time video conferencing web application built using Node.js and WebRTC.  
It aims to provide seamless video meetings, live chat, and collaboration tools in a fast and secure environment.

> ⚠️ Status: Under Development (Work in Progress)

---

### 🌟 About Meetora

Meetora is designed to be a lightweight, scalable, and developer-friendly alternative to traditional meeting platforms.  
The goal is to create a smooth and reliable real-time communication experience for teams, students, and businesses.

---

## 🎯 Planned Features

- 🔴 Real-time Video & Audio Calls
- 💬 Live Chat Messaging
- 🖥️ Screen Sharing
- 🔐 Secure Room Authentication
- 👥 Multiple Participants Support
- 📁 Meeting Room Creation & Join via Link
- 🎥 Recording (Planned)
